# 微博增强扩展 - 直接安装方式

本项目已更新为可直接在Chrome浏览器中加载的扩展形式，无需构建步骤。

## 主要功能

- 自动适应深色/浅色模式
- 弹出页查看更多评论
- 页面宽屏显示
- 自定义背景图片

## 安装方法

1. 打开Chrome浏览器，进入扩展页面：`chrome://extensions/`
2. 右上角开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹，即可安装

## 更新说明

1. 移除了对油猴API的模拟，直接使用Chrome存储API
2. 创建了native存储系统，无需额外构建步骤
3. 重构了模块加载机制，使用传统的页面脚本加载方式

## 注意事项

- 请确保安装后刷新微博页面以获得最佳体验
- 可以通过浏览器右上角的扩展图标打开设置面板
- 如需调试，可以打开开发者工具查看控制台输出

## 手动更新

当有新版本时，只需下载最新代码，然后在扩展页面点击"更新"按钮即可。
